# Taiwan Invoice
Golang SDK for Taiwan E-Invoice API

這個 SDK 是提供開發者串接以及測試台灣財政部的電子發票系統時可以更加快速以及方便所使用的。

如果有任何疑問歡迎開立 ISSUE。


SDK 功能部分分為兩個大類：
1. 電子發票應用API
2. 電子發票營業人應用 API


其中在第二類營業人應用 API 需要向財政部申請並設定 Client 端 IP 才能進行連線，這點請特別注意。
